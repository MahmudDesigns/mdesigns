# MDesigns – Creative Portfolio Website

MDesigns is a modern, responsive portfolio website built to showcase web and graphic design services with elegance and performance. It is crafted with clean HTML and styled for a professional online presence. Ideal for designers, freelancers, and creative professionals.

## 🌟 Features

* Fully responsive layout
* Smooth scrolling and animated transitions
* Clear navigation across Home, About, Services, Portfolio, and Contact pages
* Legal compliance pages: Privacy Policy, Terms of Services, Cookie Policy
* Easy to edit and deploy

## 📁 Project Structure

```
mdesigns/
├── index.html
├── about.html
├── services.html
├── portfolio.html
├── contacts.html
├── privacy-policy.html
├── terms-of-services.html
├── cookie-policy.html
└── Images/
```

## 🚀 Getting Started

1. Download or clone the repository
2. Open `index.html` in a browser to preview
3. Customize text and images to match your brand
4. Deploy to GitHub Pages, Netlify, or any static host

## ✏️ Customization

* Update images in the `Images/` folder
* Modify page content using any text or code editor
* Adjust color palette or typography via CSS
